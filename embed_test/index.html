Embed URL properties:
  • PRODUCT ID - REQUIRED Must be an integer referencing a specific product
  • CSS URL - OPTIONAL A url pointing toward a CSS file to be loaded into the page.
  • DEFAULT BACKGROUND ID - OPTIONAL an ID referencing a background to use as the default in the editor. The value relates to the index order in the backgrounds dropdown. i.e. 0 = None, 1 = Beach, etc.

The url is structured in 3 parts:

  1. The base domain - https://kitepaint.com/
  2. The root-level query parameters - ?embed=1&css-url=[CSS URL]&id=[PRODUCT ID]&default-background-id=[DEFAULT BACKGROUND ID]
  3. The application path - #!/edit/new?id=[PRODUCT ID]

A complete URL will look like: https://kitepaint.com/?embed=1&css-url=https://example.com/pretty.css&id=5&default-background-id=1#!/edit/new/?id=5
